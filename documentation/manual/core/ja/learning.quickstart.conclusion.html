Autoloading in Zend Framework

おめでとう！

最も一般的に用いられる Zend Framework コンポーネントの一部を使って、 今や非常に単純なアプリケーションを構築しました。 Zend Frameworkは、ウェブ・アプリケーションでとても一般的に必要なものを対象にする、 多くのコンポーネントを利用可能にします。それには、ウェブ・サービス、検索、 PDF の読み書き、認証、アクセス権限などがあります。 リファレンスガイド は、他のコンポーネントばかりではなく、 このクイックスタートで使用したコンポーネントについてもっと知るための良い場所です。 Zend Framework が役に立ちますように、もっと大事なことですが、楽しいものでありますように！
