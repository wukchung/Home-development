Zend_Dom_Query(日本語)

導入

Zend_Dom は、 DOM ドキュメントや構造を扱うためのツールを提供します。 現在用意しているのは Zend_Dom_Query で、これは XPath や CSS セレクタによる DOM ドキュメントの問い合わせを共通インターフェイスで行えるようにします。
