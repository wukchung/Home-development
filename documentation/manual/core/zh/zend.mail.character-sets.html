字符集

Zend_Mail不会对邮件段字符集的正确性进行检查。当Zend_Mail初始化的时候，缺省地为邮件设置了iso8859-1字符集。程序必须确保添加至邮件对象的所有段都以正确的字符集编码。在添加一个新的邮件段时，可以为每个段指定不同的字符集。

Note: 只限于文本(Text)格式
 字符集只适用于文本格式中的段。
