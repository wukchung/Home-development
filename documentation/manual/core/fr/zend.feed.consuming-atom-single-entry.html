Consommer une entrée Atom particulière

Les éléments Atom <entry> sont aussi valides tout seuls. Généralement l'URL d'une entrée constitue l'URL du flux suivie de /<idEntree>, par exemple http://atom.exemple.com/flux/1 si on reprend l'URL que nous avons utilisée ci-dessus.

Si vous lisez une entrée seule, vous obtiendrez toujours un objet Zend_Feed_Atom mais cette classe créera automatiquement un flux "anonyme" pour contenir l'entrée.

Example #1 Lire un flux Atom constitué d'une seule entrée

  1. $flux = new Zend_Feed_Atom('http://atom.exemple.com/flux/1');
  2. echo 'Le flux possède : ' . $flux->count() . ' entrée(s).';
  3.  
  4. $entree = $flux->current();

Vous pouvez aussi instancier directement la classe représentant les entrées si vous êtes sûr que vous accédez à un document contenant une seule balise <entry> :

Example #2 Utiliser directement l'objet Zend_Feed_Entry_Atom

  1. $entree = new Zend_Feed_Entry_Atom('http://atom.exemple.com/flux/1');
  2. echo $entree->title();
