Sauvegarder les changement dans un document PDF

Il y a deux méthodes qui permettent la sauvegarde dans un document PDF. Ce sont Zend_Pdf::save() et Zend_Pdf::render().

La méthode Zend_Pdf::save($filename, $updateOnly = false) sauvegarde un document dans un fichier. Si $updateOnly est à TRUE, alors seulement les nouvelles sections PDF sont ajoutées au fichier. Sinon le fichier est écrasé.

La méthode Zend_Pdf::render($filename, $newSegmentOnly = false) retourne le document PDF dans une chaîne. Si $newSegmentOnly est à TRUE, alors seulement les nouvelles sections du PDF sont retournées.

Example #1 Sauvegarder un document PDF

  1. ...
  2. // Charge un document PDF.
  3. $pdf = Zend_Pdf::load($fileName);
  4. ...
  5. // Met à jour le document
  6. $pdf->save($fileName, true);
  7. // Sauvegarde le document dans un nouveau fichier.
  8. $pdf->save($newFileName);
  9.  
  10. // Retourne le document PDF dans une string
  11. $pdfString = $pdf->render();
  12. ...
