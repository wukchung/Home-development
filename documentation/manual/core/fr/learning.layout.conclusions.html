Zend_Layout: Conclusions

Zend_Layout est une simple surcouche à Zend_View en proposant un pattern Two-Step View, offrant la fléxibilité de créer un design applicatif dans lequel le contenu est injecté.

Si vous regardez de près les exemple, vous sentirez peut-être que les fonctionnalités sont limitées : comment modifier le titre de la page, injecter un tag script optionnel ou même créer une sidebar? Ces questions concernent le concept de "Composite View" -- et trouveront réponse dans le chapitre suivant couvrant des "placeholders."
