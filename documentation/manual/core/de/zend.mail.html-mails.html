HTML E-Mail

Um einen E-Mail im HTML Format zu versenden, muss der Hauptteil der Nachricht durch die setBodyHTML() statt durch die setBodyText() gesetzt werden. Der MIME Inhaltstyp wird dann automatisch auf text/html gesetzt. Wenn du sowohl HTML als auch Text Daten verwendest, wird automatisch eine multipart/alternative MIME E-Mail erstellt:

Example #1 Versand von HTML E-Mail

  1. $mail = new Zend_Mail();
  2. $mail->setBodyText('Mein netter Test Text');
  3. $mail->setBodyHtml('Mein netter <b>Test</b> Text');
  4. $mail->setFrom('somebody@example.com', 'Ein Versender');

  6. $mail->setSubject('TestBetreff');
  7. $mail->send();
