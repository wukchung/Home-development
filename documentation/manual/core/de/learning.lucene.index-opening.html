Indezes öffnen und erstellen

Alle Index Operationen (z.B. Erstellung eines neuen Index, hinzufügen eines Dokuments zum Index, löschen eines Dokuments, suchen durch den Index) benötigen ein Index Objekt. Es kann empfangen werden indem eine der folgenden zwei Methoden verwendet wird.

Example #1 Lucene Index Erstellung

  1. $index = Zend_Search_Lucene::create($indexPath);

Example #2 Lucene Index Öffnen

  1. $index = Zend_Search_Lucene::open($indexPath);
