Die MIME Abgrenzung kontrollieren

In einer mehrteiligen Nachricht wird eine MIME Abgrenzung zum Trennen der verschiedenen Nachrichtenteile normalerweise willkürlich generiert. In einigen Fällen möchtest Du jedoch die zu verwendene MIME Abgrenzung selber angeben. Dies kann durch die setMimeBoundary() Methode erreicht werden, wie in dem folgenden Beispiel:

Example #1 Die MIME Abgrenzung ändern

  1. $mail = new Zend_Mail();
  2. $mail->setMimeBoundary('=_' . md5(microtime(1) . $someId++));
  3. // erstelle Nachricht...
