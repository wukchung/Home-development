View Platzhalter: Fazit

View Platzhalter sind eine einfache und machtvolle Methode für die Erstellung reicher Layouts für die Anwendung. Man kann iene Vielzahl von standardmäßigen Platzhaltern, wie die diskutieren verwenden ( doctype(), headTitle(), headLink(), und headScript()) oder den generischen placeholder() Helfer um Inhalte zu sammeln und auf eigene Wege darzustellen. Man kann mit deren angebotenen Funktionalität experimentieren und die betreffenden Sektionen im Referenz Handbuch besuchen um mehr über die zusätzlichen Features zu erfahren die Sie anbieten -- und darüber wie diese Features angewendet werden können um reiche Inhalten für die Leser zu erstellen.
