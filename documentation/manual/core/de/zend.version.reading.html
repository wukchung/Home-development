Die Version des Zend Frameworks erhalten

Zend_Version bietet eine Klassenkonstante Zend_Version::VERSION die einen String enthält, welcher die Versionsnummer der eigenen Installation des Zend Frameworks enthält. Zend_Version::VERSION kann zum Beispiel "1.7.4" enthalten.

Die statische Methode Zend_Version::compareVersion($version) basiert auf der PHP Funktion » version_compare(). Die Methode gibt -1 zurück wenn die angegebene Version älter als die installierte Version des Zend Frameworks ist, 0 wenn Sie identisch sind und +1 wenn die angegebene Version neuer als die Version des Zend Frameworks ist.

Example #1 Beispiel der compareVersion() Methode

  1. // gibt -1, 0 oder 1 zurück
  2. $cmp = Zend_Version::compareVersion('2.0.0');
