Einführung

Zend_Amf bietet Unterstützung für Adobe's » Action Message Format (AMF), um Kommunikation zwischen Adobe's » Flash Player und PHP zu ermöglichen. Im speziellen, bietet es ein Gateway Server Implementation für die Behandlung von Anfragen die vom Flash Player zum Server gesendet werden und die Umlegung dieser Anfragen zum Objekt, den Klasenmethoden, und möglichen Callbacks.

Die » AMF3 Spezifikation ist frei erhältlich, und fungiert als Referenz für die Typen von Nachrichten die zwischen dem Flash Player und dem Server gesendet werden.
