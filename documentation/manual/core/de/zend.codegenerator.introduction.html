Zend_CodeGenerator Beispiele

Einführung

Zend_CodeGenerator bietet die Möglichkeit jeglichen Code zu erstellen indem ein Objekt Orientiertes Interface verwendet wird, um sowohl neuen Code zu erstellen als auch bestehenden Code upzudaten. Wärend die aktuelle Implementation darin limitiert ist PHP Code zu erstellen, kann die Basisklasse einfach erweitert werden um Code Erzeugung für andere Zwecke zu bieten: JavaScript, Konfigurationsdateien, Apache VHosts, usw.

Theorie der Anwendung

Der typischste Fall ist die Instanzierung einer Code Erzeugungs Klasse und entweder der Übergabe der entsprechenden Konfiguration oder der Konfiguration nach dessen Erstellung. Um Code zu erstellen, muß man einfach das Objekt ausgeben, oder dessen generate() Methode aufrufen.

  1. // Konfiguration an den Construktor übergeben:
  2. $file = new Zend_CodeGenerator_Php_File(array(
  3.     'classes' => array(
  4.         new Zend_CodeGenerator_Php_Class(array(
  5.             'name'    => 'World',
  6.             'methods' => array(
  7.                 new Zend_CodeGenerator_Php_Method(array(
  8.                     'name' => 'hello',
  9.                     'body' => 'echo \'Hallo Welt!\';',
  10.                 )),
  11.             ),
  12.         )),
  13.     )
  14. ));
  15.  
  16. // Konfiguration nach der Initialisierung
  17. $method = new Zend_CodeGenerator_Php_Method();
  18. $method->setName('hello')
  19.        ->setBody('echo \'Hallo Welt!\';');
  20.  
  21. $class = new Zend_CodeGenerator_Php_Class();
  22. $class->setName('World')
  23.       ->setMethod($method);
  24.  
  25. $file = new Zend_CodeGenerator_Php_File();
  26. $file->setClass($class);
  27.  
  28. // Die erzeugte Datei darstellen
  29. echo $file;
  30.  
  31. // oder Sie in eine Datei schreiben:
  32. file_put_contents('World.php', $file->generate());

Beide der obigen Beispiele werden das gleiche Ergebnis darstellen:

  1. <?php
  2.  
  3. class World
  4. {
  5.  
  6.     public function hello()
  7.     {
  8.         echo 'Hallo Welt!';
  9.     }
  10.  
  11. }

Ein anderer üblicher Anwendungsfall ist die Aktualisierung von bestehendem Code -- zum Beispiel eine Methode zu einer Klasse hinzufügen. In so einem Fall, muß man zuerst den bestehenden Code betrachten, indem Reflection verwendet wird, und dann die neue Methode hinzufügen. Zend_CodeGenerator macht das sehr trivial, indem Zend_Reflection verwendet wird.

Als Beispiel nehmen wir an dass das obige in der Datei "World.php" abgespeichert wurde, und wir diese bereits inkludiert haben. Wir könnten dann das folgende tun:

  1. $class = Zend_CodeGenerator_Php_Class::fromReflection(
  2.     new Zend_Reflection_Class('World')
  3. );
  4.  
  5. $method = new Zend_CodeGenerator_Php_Method();
  6. $method->setName('mrMcFeeley')
  7.        ->setBody('echo \'Hallo, Mr. McFeeley!\';');
  8. $class->setMethod($method);
  9.  
  10. $file = new Zend_CodeGenerator_Php_File();
  11. $file->setClass($class);
  12.  
  13. // Die erzeugte Datei darstellen
  14. echo $file;
  15.  
  16. // Oder besser, es in die originale Datei zurückschreiben:
  17. file_put_contents('World.php', $file->generate());

Die resultierende Klasse würde wie folgt aussehen:

  1. <?php
  2.  
  3. class World
  4. {
  5.  
  6.     public function hello()
  7.     {
  8.         echo 'Hallo Welt!';
  9.     }
  10.  
  11.     public function mrMcFeeley()
  12.     {
  13.         echo 'Hallo, Mr. McFeeley!';
  14.     }
  15.  
  16. }
