Einführung

Zend_Application bietet eine Möglichkeit des Bootstrappings für Anwendungen für wiederverwendbare Ressourcen, Normal- und Modul-basierende Bootstrap Klassen und der Prüfung deren Abhängigkeit. Es nimmt auch auf die Einstellungen im PHP Environment acht und führt standardmäßig Autoloading durch.
