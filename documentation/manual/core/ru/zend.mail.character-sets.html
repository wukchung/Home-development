Наборы символов

Zend_Mail не проверяет правильность выбранного набора символов для частей сообщения. Набор символов для сообщения может быть задан при инстанцировании Zend_Mail. По умолчанию это iso-8859-1. Приложение должно обеспечивать кодирование с правильным набором символов всех частей, добавляемых в объект сообщения. При создании новых частей сообщения отличающаяся кодировка должна быть указана.

Note: Только для текстового формата
 Наборы символов применимы только к частям сообщения в текстовом формате.
