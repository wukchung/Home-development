Advanced Usage of Zend_Json

Основы применения

Использование Zend_Json включает в себя применение двух публичных статических (public static) методов: Zend_Json::encode() и Zend_Json::decode().

  1. <?php
  2. // Получение значения:
  3. $phpNative = Zend_Json::decode($encodedValue);
  4.  
  5. // Преобразование для возвращения клиенту:
  6. $json = Zend_Json::encode($phpNative);
  7. ?>
